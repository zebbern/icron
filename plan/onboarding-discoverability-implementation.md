# Onboarding, Discoverability & Quick Actions Implementation

## Overview

Implement user-friendly features to improve icron's first-time experience and ongoing usability.

## Tasks

### HIGH IMPACT - Onboarding

- [ ] Task 1: Guided setup wizard (`icron setup --guided`)
  - Interactive prompts for API keys
  - Channel configuration
  - Workspace setup
  - Validation at each step

- [ ] Task 2: Config validation
  - `icron validate` command
  - Real-time validation in CLI
  - Helpful error messages with suggestions

### MEDIUM - Discoverability

- [ ] Task 3: `/skills` command
  - List all available skills by category
  - Show skill descriptions
  - `/skills run [name]` for direct execution

- [ ] Task 4: Example prompts
  - "Try saying:" suggestions for new users
  - Context-aware suggestions
  - First-time user hints

### MEDIUM - Quick Actions

- [ ] Task 5: `/weather` command
  - Quick weather lookup
  - Location-aware (configurable default)
  - Forecast options

- [ ] Task 6: Message templates
  - Morning briefing template
  - Daily summary template
  - Research task template

- [ ] Task 7: Documentation and testing
